# Economic Analysis

This project uses Pandas and matplotlib to analyze and visualize economic indicators, using live data collected from FRED (Federal Reserve Economic Data). The project is contained in a Python virtual environment (venv), with all of the required packages installed using pip.

### NOTE
When exploring the main.ipynb notebook, make sure the interactive kernel is set to ./bin/python. Also, the FRED API key is inside a local file that is not tracked via git, so if you want to test the notebook, you must create an .env file with this line:

```
FRED_API_KEY='put your FRED API key here'
```
